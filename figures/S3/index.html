    S3 · Suppl Figures

    • Suppl Figures 3C was plotted in Integrative Genomics Viewer1 using somatic variant data.
    • Suppl Figures 3E was plotted using palimpsest_plotTumorHistories and palimpsest_DissectSigs functions in Palimpsest R package2. Author of Palimpsest package has provided a detailed documentation on how to run Palimpsest workflow.

    1. Robinson JT, Thorvaldsdóttir H, Winckler W, Guttman M, Lander ES, Getz G, et al. Integrative genomics viewer. Nat Biotechnol 2011;29:24–6. doi: 10.1038/nbt.1754. ↩

    2. Shinde J, Bayard Q, Imbeaud S, Hirsch TZ, Liu F, Renault V, et al. Palimpsest: an R package for studying mutational and structural variant signatures along clonal evolution in cancer. Bioinformatics 2018. doi: 10.1093/bioinformatics/bty388. ↩
